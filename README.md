# 💅 Soleil Nail Studio - Smart Clinic Management System

מערכת Full-Stack חכמה ומקיפה לניהול קליניקת קוסמטיקה וטיפוח. הפרויקט פותח במטרה להפוך תהליכים ידניים לאוטומטיים לחלוטין – החל משלב קביעת התור על ידי הלקוחה, דרך תזכורות SMS, ועד לניהול פיננסי אוטומטי של העסק.

המערכת מחולקת לשני ממשקים מרכזיים:
1. **Client App (אפליקציית לקוחות):** ממשק Web אלגנטי, נקי ומותאם למובייל המאפשר ללקוחות לקבוע תורים, להירשם לרשימות המתנה ולאשר הגעה.
2. **Admin Dashboard (מערכת ניהול):** לוח בקרה מאובטח למנהלת הקליניקה השולט על כל הנתונים בזמן אמת.

---

## ✨ פיצ'רים מרכזיים ומערכות אוטומציה

* 📅 **יומן חכם וניהול משמרות:** מערכת תורים שמונעת כפילויות, מחשבת זמני טיפול דינמיים (כולל חסימת שעות עבר) ותומכת בפתיחת יומן עתידי בקליק.
* 🤖 **אוטומציית הודעות SMS (אינטגרציית Pulseem):** 
  * שליחת קוד אימות (OTP) בכניסה למערכת.
  * תזכורות אוטומטיות ללקוחות יום לפני התור בשעה 20:00 (Cron Job).
  * הודעת SMS אוטומטית לרשימת ההמתנה ברגע שתור מתבטל.
  * הפצת הודעות תפוצה לכלל הלקוחות ישירות ממסך הניהול.
* 📲 **התראות Telegram בזמן אמת:** אינטגרציה עם בוט טלגרם ששולח למנהלת עדכונים על תורים חדשים, בקשות המתנה ולקוחות שנרשמו, כולל כפתורי "אישור מהיר" מתוך הצ'אט.
* 💰 **מערכת פיננסית חכמה:** מעקב הכנסות והוצאות חודשי. הכנסות מתורים שהסתיימו ואושרו נרשמות **אוטומטית** בסוף היום ללא מגע יד אדם.
* 🎁 **ניהול טיפולים דינמי:** שליטה מלאה על קטלוג הטיפולים – הוספה, עריכה, שינוי סדר התצוגה, ותמחור מותאם ("החל מ-").
* 👥 **רשימת המתנה (Waitlist):** מנגנון המאפשר ללקוחות להירשם לשעות עמוסות ולקבל התראה מידית ברגע שמתפנה מקום.

## 🛠️ טכנולוגיות וארכיטקטורה

הפרויקט בנוי בארכיטקטורת Serverless המבטיחה זמינות גבוהה, אבטחת מידע וביצועים מעולים.

**Frontend:**
* HTML5, CSS3
* Vanilla JavaScript (ES6+)
* עיצוב רספונסיבי מותאם אישית (Mobile-First)

**Backend & Database (Firebase):**
* **Firestore (NoSQL):** ניהול וסנכרון נתונים בזמן אמת (Realtime Subscriptions).
* **Cloud Functions (Node.js):** ניהול הלוגיקה העסקית שרצה מאחורי הקלעים (שליחת SMS, אוטומציות כספים, התראות).
* **Cloud Scheduler:** תזמון משימות אוטומטיות יומיות ושעתיות.
* **Firebase Hosting:** אחסון והגשת האפליקציה במהירות.

**Integrations (APIs):**
* Telegram Bot API
* Pulseem SMS API
