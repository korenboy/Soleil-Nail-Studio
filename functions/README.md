# 💅 Soleil Nail Studio - Clinic Management System

מערכת מלאה ומתקדמת לניהול קליניקה לטיפוח הציפורניים. 
המערכת כוללת אפליקציית צד-לקוח חכמה לקביעת תורים, ומערכת צד-אדמין (מנהלת) לניהול העסק.

## ✨ פיצ'רים מרכזיים
* **מערכת תורים חכמה:** קביעת תורים לפי יומן ומשמרות המוגדרות מראש.
* **ניהול טיפולים דינמי:** הוספה, הסרה ושינוי סדר הופעת הטיפולים, כולל תמחור "החל מ-".
* **אוטומציית הודעות:** תזכורות אוטומטיות ללקוחות ב-SMS דרך מערכת Pulseem.
* **התראות מנהלת:** חיבור לבוט Telegram שמתריע בזמן אמת על בקשות לתורים ולקוחות חדשים.
* **ניהול פיננסי:** מעקב הכנסות והוצאות.
* **רשימת המתנה דינמית:** לקוחות יכולות להירשם לזמנים עמוסים.

## 🛠️ טכנולוגיות בשימוש
* **Frontend:** HTML5, CSS3, Vanilla JavaScript.
* **Backend:** Node.js (Firebase Cloud Functions).
* **Database:** Firebase Firestore (NoSQL).
* **Hosting:** Firebase Hosting.
